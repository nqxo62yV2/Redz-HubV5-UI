## 🇬🇧 English

### Overview
Slightly reworked version of REDZ Library, focused on visual adjustments and minor fixes. Core logic unchanged.

### Changes
- **Transparency:** `BackgroundTransparency` 0.03 → 0.35 (menu is now more see-through)
- **Code structure:** added section comments for readability
- **Fix:** when shrinking the menu, the logo/search bar no longer overflow outside the menu
- **Fix:** search bar can now find all features across all tabs

### Compatibility
- Menu structure unchanged
- Logic unchanged
- Existing setups/integrations should keep working

### Note
No menu elements were removed, no logic was changed, to keep everything running.

## 🇹🇭 ภาษาไทย

### ภาพรวม
เวอร์ชันปรับปรุงเล็กน้อยของ REDZ Library เน้นการปรับแต่งด้านภาพและแก้ไขจุดเล็กๆ น้อยๆ ลอจิกหลักไม่มีการเปลี่ยนแปลง

### การเปลี่ยนแปลง
- **ความโปร่งใส:** `BackgroundTransparency` 0.03 → 0.35 (เมนูโปร่งใสมากขึ้น)
- **โครงสร้างโค้ด:** เพิ่มคอมเมนต์แบ่งส่วนเพื่อให้อ่านง่ายขึ้น
- **แก้ไข:** เมื่อย่อเมนู โลโก้/แถบค้นหาจะไม่ล้นออกนอกเมนูอีกต่อไป
- **แก้ไข:** แถบค้นหาสามารถค้นหาฟีเจอร์ทั้งหมดในทุกแท็บได้แล้ว

### ความเข้ากันได้
- โครงสร้างเมนูไม่เปลี่ยนแปลง
- ลอจิกไม่เปลี่ยนแปลง
- การตั้งค่า/การเชื่อมต่อเดิมควรยังคงทำงานได้ปกติ

### หมายเหตุ
ไม่มีการลบองค์ประกอบใดๆ ออกจากเมนู ไม่มีการเปลี่ยนลอจิก เพื่อความเสถียรของสคริปต์
